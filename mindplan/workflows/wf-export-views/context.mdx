# Export graph views

Deterministic Mermaid/DOT projection of the graph via export_mindplan_view and the view CLI.

## Execution Logic

1. Load graph from territory.
2. Optionally filter to focus + 1-hop; hide retired unless `include_retired`.
3. Render Mermaid or DOT typed-DAG (workflow instances per journey).
4. Exposed as MCP `export_mindplan_view` and CLI `view`/`export`.

Implemented in `src/view.ts`.
