# Project init

Scaffold consumer mindplan/ plus agent playbook, define-entities skill, MCP example, integrations, and AGENTS.md.

## Execution Logic

1. Run `node dist/index.js init` (or `mindplan-mcp init`) from project root.
2. Creates `mindplan/` type dirs and installs from `templates/agent/`:
   - playbook, define-entities skill, mcp.json.example, integrations guides
3. Creates root `AGENTS.md` only when missing.
