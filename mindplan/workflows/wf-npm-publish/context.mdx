# Publish to npm

Publish mindplan-mcp to npm so consumer projects can install without building from source.

## Execution Logic

1. Confirm package metadata (`name`, `files`, `bin`, engines).
2. Run `npm test` and `npm run build`.
3. Publish `mindplan-mcp` to the public registry.
4. Update README quick start from “install from source” to `npx`/`npm i -g`.
