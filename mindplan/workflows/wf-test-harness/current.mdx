# Test harness

Smoke and view regression tests that exercise graph assembly and Mermaid/DOT export.

## Execution Logic

- `npm test` builds then runs `scripts/view.test.mjs` and `scripts/smoke.mjs`.
- Smoke covers init, node lifecycle, compiler blocks, and view export against a temp root.
- View test asserts deterministic Mermaid/DOT shape helpers.
