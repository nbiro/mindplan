# Graph search

## Purpose

Infra — orientation ranking and 1-hop neighborhood expansion for `find_related_nodes` / `orient_for_work`. Keeps retired nodes out of default ranking so agents focus on the live architecture.

## Shared Substrate Spec

Package: `src/foundations/f-graph-search/`

- `search.ts` — `tokenize` / `scoreNode` / `rankNodes`; `expandNeighborhood`; `findRelatedNodes` (focus + linked summaries); default/max find limits; excludes `deprecated`/`cancelled` nodes and closed Bugs from ranking by default (aligned with map export policy)

Depends on: `f-domain-model`.

## Acceptance Criteria

- Query ranking returns a focus plus 1-hop neighborhood summaries
- Retired nodes (`deprecated`, `cancelled`, closed Bugs) are excluded from default orientation ranking
- Limits respect DEFAULT_FIND_LIMIT / MAX_FIND_LIMIT
- Verified via `npm test` (`search.test.mjs`)
