# Territory filesystem store

Persists MindPlan nodes as context.mdx under mindplan/, including scaffold, frontmatter edges, init, and graph assembly.

## Infrastructure Spec

Implemented in `src/store.ts`:

- Discover nodes/edges from `mindplan/{journeys,foundations,workflows,bugs}/`
- Parse/patch YAML frontmatter (state, timestamps, edge arrays)
- Scaffold entity folders + `context.mdx` + `attachments/`
- `initProject` and agent asset installers (playbook, skill, integrations, `AGENTS.md`)
- Unchecked checkbox counter for Completion Check
- Root via `MINDPLAN_ROOT` or cwd
