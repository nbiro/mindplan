# View projection

## Purpose

Infra — project the MindPlan graph into Mermaid or DOT for humans, and persist the canonical Mermaid map at `mindplan/map.md` after mutations. Agents must treat MCP export/`find_related_nodes` as authority, not the map file alone.

## Shared Substrate Spec

Package: `src/foundations/f-view-projection/`

- `view.ts` — `prepareViewGraph` (optional focus, hide retired); `graphToMermaid` / `graphToDot`; `exportMindPlanView`; `persistMindPlanMap` / `mindPlanMapPath` (`MAP_FILENAME = map.md`)

Depends on: `f-domain-model`.

## Acceptance Criteria

- [x] Mermaid and DOT exports render nodes, edges, and states
- [x] Focused and full-graph views are supported
- [x] Successful graph mutations can refresh `mindplan/map.md`
- [x] Verified via `npm test` (`view.test.mjs`)
